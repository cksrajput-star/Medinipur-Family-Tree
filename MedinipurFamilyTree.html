Sharing & Saving Guide

This Family Tree application is 100% standalone and does not require any external Excel sheets to run. All data is embedded directly in this HTML file.

How to Save Changes

Web browsers prevent web pages from editing files directly on your hard drive for safety. When you add, edit, or delete members:

  1. Your changes are saved temporarily in your browser's local cache.
  2. To make them permanent, click "Export Standalone HTML" in the sidebar. This downloads a new MedinipurFamilyTree.html containing all your updates.
  3. Replace your original HTML file with the newly downloaded file.

How to Share with Family

This is a portable app that can be shared instantly:

  • Send the latest MedinipurFamilyTree.html file to your family members via WhatsApp, Email, or Google Drive.
  • They can open the file directly on any phone, tablet, or computer browser—no setups or apps required!
  • If they make changes, they can download the updated file using the same export button and send it back to you.
